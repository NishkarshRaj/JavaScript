Negative Slicing 

If a parameter is negative, the position is counted from the end of the string.
It extracts from back to front.

The Start is -(index of first extraction letter when index starts from zero at right) and same is End.
In positive slicing, we were giving end = endindex + 1
In negative slicing, we give start = startindex - 1;
